## 下载

### Docker

```
docker pull apachecn0/tutorialsteacher-zh
docker run -tid -p <port>:80 apachecn0/tutorialsteacher-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g tutorialsteacher-zh
tutorialsteacher-zh <port>
# 访问 http://localhost:{port} 查看文档
```
